# Number-to-words
Convert numbers into words easily.

# Features
* Faster and light weight.
* Easy to use.
* Compatible with any php based projects.

# Requirements
* PHP Version 5.5+

# Easy Installation
* Run the command
 `composer require nahid/number-to-words`
 * autoload file 
 `require 'vendor/autoload.php'`
